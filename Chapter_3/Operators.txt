[==] :-> Double equal too is use for check equality of two value

[=] :-> single equal to is use  for assign value to variable( Assignment operator )

[>=] :-> Greater than or equal to is use for check greater than or equal to two

[<=] :-> Less than or equal to is use for check less than or equal to two value

[!=] :-> Not equal to is use for check not equal to two value
 
[>] :-> Greater than is use for check greater than two value

[<] :-> Less than is use for check less than two value

These all operators are used in if , if else ,  if else if , switch case , while , for loop etc.
In C-programming language.
